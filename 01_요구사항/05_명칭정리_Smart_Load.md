# 명칭 정리 - Smart Load

작성일: 2026-04-07

## 1. 목적

기존에 `지선차단기`로 부르던 장치의 명칭을 `Smart Load`로 통일하기 위한 기준 문서이다.

## 2. 명칭 기준

- 기존 명칭: `지선차단기`
- 변경 명칭: `Smart Load`

## 3. 문서 사용 원칙

- 대외 설명, 보고, 회의자료에서는 `Smart Load`를 우선 사용한다.
- 내부 검토 문서에서 기존 명칭이 남아 있는 경우 아래처럼 병기한다.

권장 표기:

- `Smart Load(기존 지선차단기)`
- `Smart Load 장치`

## 4. 현재 해석

현재까지 작성된 다수 문서는 기존 명칭 `지선차단기` 기준으로 작성되어 있다.  
이후 신규 문서는 `Smart Load` 기준으로 작성하고, 기존 문서는 순차적으로 용어를 맞추는 방향으로 진행한다.

## 5. 참고

- SL = `Smart Load`
- 향후 다이어그램, 회의자료, 보고자료에서도 `Smart Load` 명칭을 우선 적용한다.
