## 3. 선정 기준

MCU와 계량칩은 아래 기준으로 판단한다.

- 가격
- 게런티
- 단종 여부
- 사이즈
- 딜리버리
- 기능 구현 가능 여부

실무적으로는 아래처럼 읽는다.

- 가격: 양산 단가와 시제품 단가 모두 확인
- 게런티/단종 여부: 현재 생산중인지, 공급 지속성이 있는지
- 사이즈: PCB 면적과 배치 난이도
- 딜리버리: 현재 재고와 리드타임
- 기능 구현 가능 여부: 현재 정의된 단상 지선차단기 기능 충족 여부

## 4. 이번 검토의 결론

### 4.1 MCU 선정

- 선정: `STM32G071CBT6`

판단 이유:

- STM32CubeMX 사용 가능
- USART 4개로 BL0942, RS-485, 서비스 UART 구성 가능
- 48핀으로 보드 크기에 유리
- 1차 제품 기능 범위에 충분

### 4.2 계량칩 선정

- 선정: `BL0942`

판단 이유:

- 단상 전압/전류/유효전력/전력량/주파수/과전류 검출 가능
- 가격 경쟁력 우수
- 패키지가 작음
- 1차 제품 범위에 적합
