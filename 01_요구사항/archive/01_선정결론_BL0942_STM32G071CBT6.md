# 선정 결론

작성일: 2026-04-03

## 1. 최종 선정 칩

- 계량칩: `BL0942`
- MCU: `STM32G071CBT6`

## 2. 선정 결과 요약

- `MAX` 칩은 초기 검토 대상이었으나 최종 선정에서는 탈락했다.
- `STM32F107VCT6`는 후보 검토 단계에서는 의미가 있었으나, 최종 선정 칩은 아니다.
- 현재 기준으로 모든 후속 문서는 `BL0942 + STM32G071CBT6`를 기준으로 진행한다.

## 3. 선정 사유

### 3.1 BL0942

- 단상 1차 기능 범위 충족
- 가격 경쟁력 우수
- 패키지 작음
- BL0942 기반으로 1차 제품을 빠르게 구성 가능

### 3.2 STM32G071CBT6

- 48핀으로 보드 구성에 유리
- USART 4개로 통신 여유 충분
- 메모리와 기능이 1차 제품에 적절
- STM32CubeMX 사용 가능
- longevity 관점에서도 유리

## 6. 초기 검토 문서 처리 원칙

초기 검토 문서는 삭제하지 않고 보관한다.

- `MAX` 기반 문서: 초기 검토 후 탈락 이력
- `STM32F107` 기반 문서: 대안 검토 이력

즉, 향후 기준 문서는 아니고 참고용 보관 문서로만 본다.
