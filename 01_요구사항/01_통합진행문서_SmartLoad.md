# Smart Load 통합 진행 문서

작성일: 2026-04-08

## 1. 문서 목적

본 문서는 Smart Load 개발 진행을 위해 필요한 내용을 하나로 묶은 기준 문서이다.

정리 순서는 아래와 같다.

1. Smart Load 기능을 `STM32L053R8T6`가 구현 가능한지 비교
2. 가격 / Delivery / 구매성 정리
3. 계량칩 관련 일정 관리
4. `LCU 보드 사용 가능 여부` 확인 후 데모 테스트 선실행
5. 기능 및 핀 선정 후 HW팀 전달
6. 4번 기준 코드 개발 및 데모 테스트
7. `STM32L053R8T6 + ATM90E26-YU` 보드 기준 코드 개발 및 데모 테스트
8. 기능 추가

현재 기준 칩 조합은 아래와 같다.

- MCU: `STM32L053R8T6`
- 계량칩: `ATM90E26-YU`

## 4. 현재 기준 진행 순서

1. 최종 칩 선정 확정
2. 계량칩 기능 / 구매성 정리
3. `LCU 보드 사용 가능 여부` 확인
4. `LCU 보드` 가능 시 `LCU + SSOP 보드` 데모 테스트
5. 불가 시 `eval 보드 + SSOP 보드` 데모 테스트
6. 기능 및 핀 선정
7. HW팀 전달 문서 작성 및 전달
8. 데모 기준 코드 개발 및 테스트
9. `STM32L053R8T6 + ATM90E26-YU` 보드 기준 코드 개발
10. 최종 데모 테스트
11. 기능 추가

## 5. 결론

현재 Smart Load 개발은 `STM32L053R8T6 + ATM90E26-YU` 기준으로 진행한다.

핵심은 바로 최종 보드 개발로 들어가기보다, 먼저 `LCU 보드 사용 가능 여부`를 확인하고 가능 시 `LCU + 계량칩 SSOP 보드`, 불가 시 `기존 eval 보드 + 계량칩 SSOP 보드`로 데모 테스트를 실행해 계량칩과 코드 구조를 선검증하는 것이다.
