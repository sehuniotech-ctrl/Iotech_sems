# Smart Load 기능 정의 및 구현 범위

## 1. 1차 제품 기준

- 단상 Smart Load
- 계량칩: `ATM90E26-YU`
- MCU: `STM32L053R8T6`

## 2. 1차 구현 기능

- 전압
- 전류
- 주파수
- 유효전력
- 무효전력
- 피상전력
- 역률
- 전력량
- 과전류 차단
- 과전압 차단
- RS-485 통신
- 날짜/시간 포함 전송
- 대기전력 차단
- 스케줄 복구 / 원격 복구

## 3. 1차 제외 기능

- 누전차단
- 영상전류
- 저항성 누설전류
- 용량성 누설전류
- 고조파
- 아크전류
