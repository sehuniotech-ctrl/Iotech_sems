# ATM90E26 기준 계량 항목 재정리

작성일: 2026-04-07
기준 칩: `ATM90E26`

## 1. 목적

기존 `BL0942` 기준으로 정리했던 계량 가능 항목을 `ATM90E26` 기준으로 다시 정리한다.

## 2. 정리 기준

- `직접 가능`: ATM90E26 레지스터 또는 출력 기능으로 직접 취득 가능
- `MCU 계산/판단 가능`: ATM90E26의 직접 계측값을 바탕으로 MCU가 계산하거나 임계값 비교로 구현 가능
- `사실상 어려움`: ATM90E26만으로는 부족하고 별도 센서, 별도 계측 경로 또는 다른 칩이 필요

## 3. ATM90E26로 직접 가능 항목

- 전압
- 주파수
- 부하 전류
- 유효전력
- 유효전력량
- 무효전력
- 무효전력량
- 피상전력
- 역률
- 위상각

## 4. MCU 계산 또는 판단으로 가능한 항목

- 과전류
  - ATM90E26의 전류 측정값을 기준으로 MCU가 임계값 비교
- 과전압
  - ATM90E26의 전압 측정값을 기준으로 MCU가 임계값 비교
- 차단 판단
  - 과전류, 과전압, 대기전력 조건 등을 MCU 로직으로 판단
- 이벤트 전송
  - MCU와 통신부가 처리
- 날짜/시간 포함 전송
  - MCU와 RTC 또는 상위 장치 시간 동기화로 처리
- MAC/IP 포함 전송
  - MCU/통신부가 처리

## 5. ATM90E26로 직접 지원되지 않는 항목

- 고조파 발생현황
- 누전차단
- 영상전류
- 저항성 누설전류
- 용량성 누설전류
- 아크전류
- 제품 내부온도

## 6. 항목별 판정표

| 항목 | ATM90E26 가능 여부 | 방식 | 비고 |
|---|---|---|---|
| 전압 | 가능 | 직접 측정 | `Urms` 계열 |
| 주파수 | 가능 | 직접 측정 | 주파수 측정 지원 |
| 부하 전류 | 가능 | 직접 측정 | `Irms` 계열 |
| 전력량 | 가능 | 직접 측정/누적 | 유효/무효 에너지 누적 가능 |
| 과전류 | 가능 | MCU 판단 | 직접 보호 기능보다 임계값 비교 방식이 적합 |
| 유효전력 | 가능 | 직접 측정 | 평균 유효전력 지원 |
| 무효전력 | 가능 | 직접 측정 | 평균 무효전력 지원 |
| 피상전력 | 가능 | 직접 측정 | 평균 피상전력 지원 |
| 역율 | 가능 | 직접 측정 | PF 지원 |
| 고조파 발생현황 | 어려움 | 사실상 별도 분석 필요 | 하모닉 분석 엔진 없음 |
| 누전차단 | 어려움 | 별도 센서/회로 필요 | ZCT 경로 필요 |
| 영상전류 | 어려움 | 별도 센서/회로 필요 | N선 채널은 ZCT 대체 아님 |
| 저항성 누설전류 | 어려움 | 별도 센서/위상분석 필요 | ZCT 기반 별도 처리 필요 |
| 용량성 누설전류 | 어려움 | 별도 센서/위상분석 필요 | ZCT 기반 별도 처리 필요 |
| 아크전류 | 어려움 | 별도 회로/알고리즘 필요 | 고주파 아크 검출 기능 없음 |
| 제품 내부온도 | 어려움 | 별도 센서 필요 | 내부 온도 계측 기능 없음 |
| 날짜/시간 포함 전송 | 가능 | MCU 처리 | 칩 자체 기능은 아님 |
| MAC/IP 포함 전송 | 가능 | MCU/통신부 처리 | 칩 자체 기능은 아님 |

## 7. ATM90E26 기준으로 달라지는 점

- `BL0942` 대비 `무효전력`, `피상전력`, `역률`, `위상각`을 직접 계측 레벨로 가져가기 쉬워진다.
- 단상 전력 품질 기본 항목은 `ATM90E26` 쪽이 더 풍부하다.
- 다만 `고조파`, `누설전류`, `아크`, `내부온도`는 여전히 별도 경로가 필요하다.

## 8. 주의사항

- `ATM90E26`은 `N line current` 채널을 제공하지만, 이것을 바로 `영상전류` 또는 `누설전류` 계측으로 동일하게 보면 안 된다.
- `전압 sag` 경고와 `WarnOut/IRQ` 기능은 지원하지만, 이것이 곧 과전압/과전류 보호 기능을 직접 대체하는 것은 아니다.
- 과전류, 과전압 차단은 ATM90E26 값 기반으로 MCU가 판단하는 구조가 현실적이다.

## 9. 결론

`ATM90E26`은 단상 지선차단기에서 필요한 기본 계량 항목에 대해 `BL0942`보다 넓은 직접 계측 범위를 제공한다.

특히 아래 항목은 `ATM90E26` 쪽이 유리하다.

- 무효전력
- 피상전력
- 역률
- 위상각

반면 아래 항목은 여전히 별도 대응이 필요하다.

- 고조파
- 누설전류
- 영상전류
- 아크전류
- 내부온도

따라서 `ATM90E26`은 단상 계량 기능 강화에는 적합하지만, `누전/아크/고조파 진단형` 기능까지 단독으로 해결하는 칩은 아니다.
