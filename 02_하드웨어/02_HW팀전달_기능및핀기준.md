# HW팀 전달 기능-핀 기준표

작성일: 2026-04-08

## 1. 기준 칩

| 구분 | 부품명 | 비고 |
|---|---|---|
| MCU | `STM32L053R8T6` | `LQFP64` 기준 |
| 계량칩 | `ATM90E26-YU` | 단상 계량칩 |
| 제품명 | `Smart Load` | 1차 / 2차 제품 기준 |

## 2. 1차 제품 기준

### 2.1 기능 범위

| 구분 | 항목 | 적용 여부 | 비고 |
|---|---|---|---|
| 계측 | 전압 / 전류 / 주파수 | 적용 | ATM90E26 직접 계측 |
| 계측 | 유효전력 / 무효전력 / 피상전력 / 역률 / 전력량 | 적용 | ATM90E26 직접 계측 + MCU 전송 |
| 보호 | 과전류 차단 / 과전압 차단 | 적용 | MCU 임계값 판단 |
| 통신 | RS-485 polling | 적용 | DCU 통신 |
| 제어 | 릴레이 ON/OFF / 대기전력 차단 / 스케줄 복구 / 원격 복구 | 적용 | 펌웨어 로직 |
| 확장 | 상태 LED | 미적용 | 2차 반영 |
| 확장 | 누전 / 영상전류 | 미적용 | ZCT 필요 |
| 확장 | 고조파 / 아크 | 미적용 | 별도 회로 필요 |

### 2.2 인터페이스 및 권장 핀

| 연결 구간 | 방식 | 권장 핀 | 비고 |
|---|---|---|---|
| MCU ↔ ATM90E26 | `SPI1` | `PA5, PA6, PA7, PB6, PB7` | `SCK/MISO/MOSI/CS/RESET` |
| MCU ↔ DCU | `USART2 + RS-485` | `PA2, PA3, PA1` | `TX/RX/DE-RE` |
| MCU ↔ Relay | GPIO | `PB0, PB1` | `Drive/Feedback` |
| 서비스 UART | `USART1` | `PA9, PA10` | 디버그 / 로그 |
| ATM90 확장핀 | `GPIO/EXTI` | `PB10~PB14` | `IRQ/WarnOut/ZX/CF1/CF2` |

## 3. 2차 제품 기준

### 3.1 기능 범위

| 구분 | 항목 | 적용 여부 | 비고 |
|---|---|---|---|
| 계측 | 전압 / 전류 / 주파수 | 적용 | 1차 유지 |
| 계측 | 유효전력 / 무효전력 / 피상전력 / 역률 / 전력량 | 적용 | 1차 유지 |
| 보호 | 과전류 차단 / 과전압 차단 | 적용 | 1차 유지 |
| 통신 | RS-485 polling | 적용 | 1차 유지 |
| 제어 | 릴레이 ON/OFF / 대기전력 차단 / 스케줄 복구 / 원격 복구 | 적용 | 1차 유지 |
| 표시 | 상태 LED | 적용 | RUN / FAULT 등 |
| 확장 | `IRQ`, `WarnOut`, `ZX`, `CF1`, `CF2` 활용 | 적용 검토 | ATM90 보조 출력 활용 |
| 확장 | 내부 온도 | 적용 검토 | 별도 센서 또는 MCU 센서 |
| 확장 | 누전 / 영상전류 | 적용 검토 | ZCT 추가 필요 |
| 확장 | 고조파 / 아크 | 적용 검토 | 별도 회로 및 알고리즘 필요 |

### 3.2 인터페이스 및 권장 핀

| 연결 구간 | 방식 | 권장 핀 | 비고 |
|---|---|---|---|
| MCU ↔ ATM90E26 | `SPI1` | `PA5, PA6, PA7, PB6, PB7` | `SCK/MISO/MOSI/CS/RESET` |
| MCU ↔ DCU | `USART2 + RS-485` | `PA2, PA3, PA1` | `TX/RX/DE-RE` |
| MCU ↔ Relay | GPIO | `PB0, PB1` | `Drive/Feedback` |
| 상태 LED | GPIO | `PC8, PC9` | `RUN/FAULT` |
| 서비스 UART | `USART1` | `PA9, PA10` | 디버그 / 로그 |
| ATM90 확장핀 | `GPIO/EXTI` | `PB10~PB14` | `IRQ/WarnOut/ZX/CF1/CF2` |

## 4. HW팀 확인 요청 항목

| 항목 | 확인 내용 |
|---|---|
| MCU 패키지 | `STM32L053R8T6 LQFP64` 확정 여부 |
| 계량칩 인터페이스 | `SPI1` 적용 가능 여부 |
| RS-485 | 트랜시버 선정 및 `DE/RE` 구조 |
| Relay | 구동 회로 / 피드백 회로 적용 가능 여부 |
| ATM90 보조핀 | `IRQ`, `WarnOut`, `ZX`, `CF1`, `CF2` 보드 반영 여부 |
| 상태 LED | 2차 제품에서 `PC8`, `PC9` 활용 가능 여부 |
| 보호회로 | MCU 전원 / 계량칩 입력 / 서지 보호 구조 |

## 5. 비고

- 본 문서는 HW팀 전달용 기준 문서로 계속 업데이트한다.
- 현재 권장 구조는 `SPI1 + USART2 + GPIO Relay`이다.
- 상태 LED는 `1차 제품 미적용`, `2차 제품 적용 검토` 기준으로 관리한다.
