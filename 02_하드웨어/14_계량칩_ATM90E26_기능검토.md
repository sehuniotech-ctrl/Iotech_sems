# 계량칩 ATM90E26 기능 검토

작성일: 2026-04-07
대상 칩: `ATM90E26`

## 1. 요약

`ATM90E26`은 단상 고성능 전력 계량칩으로, 기존 `BL0942` 대비 직접 지원하는 계량 항목이 더 많다.

직접 확인 가능한 핵심 항목:

- 전압 RMS
- 전류 RMS
- 유효전력
- 무효전력
- 피상전력
- 역률
- 위상각
- 유효/무효 에너지
- 주파수

## 2. 하드웨어 관점 핵심

- 단상 2선/3선 계량 구조에 적합
- SPI/UART 인터페이스 지원
- `WarnOut`, `IRQ`, `ZX` 출력 지원
- 전압 sag 감지 지원
- N line current 채널 지원

## 3. 구현 시 주의사항

- 과전류 차단은 별도 fast RMS 보호칩처럼 보는 것보다, 측정값을 MCU에서 비교하는 구조가 적합하다.
- 과전압 차단도 전압 sag 반대 방향의 직접 보호 기능으로 보기보다, 전압 측정값을 MCU에서 비교하는 구조가 적합하다.
- N line current 채널은 anti-tampering/추가 전류 감시용으로 해석하는 것이 안전하며, ZCT 기반 누설전류 측정과 동일시하면 안 된다.

## 4. 최종 판단

- 단상 지선차단기의 기본 계량 성능 강화 목적에는 적합
- `무효전력`, `피상전력`, `역률` 직접 취득이 가능해 문서 정리가 쉬움
- `고조파`, `누전`, `아크`까지 요구하면 별도 센서와 회로가 필요
