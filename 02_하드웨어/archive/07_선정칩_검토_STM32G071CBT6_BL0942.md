# 선정 칩 검토

작성일: 2026-04-03  
대상: `STM32G071CBT6`, `BL0942`

## 1. 선정 결과

- MCU: `STM32G071CBT6`
- 계량칩: `BL0942`

## 2. 선정 기준

- 가격
- 게런티 / 단종 여부
- 사이즈
- 딜리버리
- 기능 구현 가능 여부

## 3. 1차 제품 기능 범위

포함:

- 전압 RMS
- 전류 RMS
- 유효전력
- 유효전력량
- 주파수
- 과전류 차단
- 차단기 상태 관리
- RS-485 통신

제외:

- ZCT
- 누설전류
- IGR / IGC
- 고조파
- 아크

## 4. MCU: STM32G071CBT6

### 4.1 요약 판단

- 종합 판단: `적합`

### 4.2 평가

| 항목 | 평가 | 내용 |
|---|---|---|
| 가격 | 양호 | LCSC 기준 `1k=$1.2843` |
| 게런티/단종 | 양호 | ST Active / volume production, 10년 longevity 계열 |
| 사이즈 | 양호 | `LQFP-48` |
| 딜리버리 | 양호 | LCSC 기준 재고 `6,397` |
| 기능 구현 | 양호 | USART 4개, Flash 128KB, SRAM 36KB |

### 4.3 판단 근거

- BL0942 통신과 RS-485 통신을 분리 가능
- 서비스 UART 또는 디버그 UART를 추가로 둘 수 있음
- 48핀이라 회로 구성과 보드 면적에 유리
- STM32CubeMX 사용 가능

## 5. 계량칩: BL0942

### 5.1 요약 판단

- 종합 판단: `적합`

### 5.2 평가

| 항목 | 평가 | 내용 |
|---|---|---|
| 가격 | 매우 양호 | LCSC 기준 `1k=$0.2248` |
| 게런티/단종 | 보통 이상 | 공식 제품 페이지 및 데이터시트 확인 가능 |
| 사이즈 | 양호 | `TSSOP-14` |
| 딜리버리 | 양호 | LCSC 기준 재고 `3,997` |
| 기능 구현 | 양호 | 단상 계량 필수 기능 충족 |

### 5.3 지원 범위

- 전압 RMS
- 전류 RMS
- 유효전력
- 유효전력량
- 빠른 전류 RMS
- 과전류 검출 보조
- 주파수 측정
- UART / SPI 통신

## 6. 선정 조합의 기능 구현 가능 여부

| 기능 | 구현 가능 여부 | 구현 방식 |
|---|---|---|
| 전압 계측 | 가능 | BL0942 직접 계측 |
| 전류 계측 | 가능 | BL0942 직접 계측 |
| 유효전력 | 가능 | BL0942 직접 계측 |
| 유효전력량 | 가능 | BL0942 계측 + MCU 누적 관리 |
| 주파수 | 가능 | BL0942 직접 계측 |
| 과전류 판단 | 가능 | BL0942 빠른 전류 RMS + MCU 임계 판단 |
| 차단기 제어 | 가능 | STM32 GPIO 제어 |
| 차단기 상태 읽기 | 가능 | STM32 GPIO 입력 |
| RS-485 통신 | 가능 | STM32 UART 사용 |
| 이벤트 로그 | 가능 | STM32 메모리/저장 정책 사용 |
| 피상전력/역률 | 가능 | MCU 계산 |
| ZCT 기반 누설전류 | 이번 1차 제외 | 별도 회로 필요 |
| 고조파 분석 | 이번 1차 제외 | 별도 분석 경로 필요 |
| 아크 검출 | 이번 1차 제외 | 별도 설계 필요 |

## 7. 결론

`STM32G071CBT6 + BL0942` 조합은 단상 지선차단기 1차 제품 범위에 적합하다.

특히 아래 점에서 유리하다.

- 원가 경쟁력
- 보드 크기 축소
- CubeMX 기반 개발 편의성
- 단상 기본 계량 기능 충족
