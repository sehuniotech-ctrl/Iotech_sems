# STM32G071CBT6 + BL0942 권장 핀맵 초안

작성일: 2026-04-03

## 1. 기준

- MCU: `STM32G071CBT6`
- 계량칩: `BL0942`
- 통신: `RS-485`
- 개발환경: `STM32CubeMX`

## 2. 필수 핀

### 2.1 시스템

| 핀 | 기능 | 비고 |
|---|---|---|
| `VDD`, `VSS` | 디지털 전원 | 필수 |
| `VDDA`, `VSSA` | 아날로그 전원 | 필수 |
| `NRST` | 외부 리셋 | 필수 |
| `BOOT0` | 부트 모드 | 풀다운 권장 |
| `PA13` | SWDIO | 디버그 필수 |
| `PA14` | SWCLK | 디버그 필수 |
| `PF0` | HSE_IN | HSE 사용 권장 |
| `PF1` | HSE_OUT | HSE 사용 권장 |

### 2.2 기능

| 핀 | 기능 | 비고 |
|---|---|---|
| `PA2` | BL0942 UART TX | USART2 권장 |
| `PA3` | BL0942 UART RX | USART2 권장 |
| `PB10` | RS-485 UART TX | USART3 권장 |
| `PB11` | RS-485 UART RX | USART3 권장 |
| `PB12` | RS-485 DE/RE | GPIO |
| `PC6` | Relay Drive | GPIO 출력 |
| `PC7` | Relay Feedback | GPIO 입력 |
| `PC8` | RUN LED | GPIO 출력 |
| `PC9` | TRIP/FAULT LED | GPIO 출력 |

## 3. 권장 핀

| 핀 | 기능 | 비고 |
|---|---|---|
| `PA9` | 서비스 UART TX | 로그/점검용 |
| `PA10` | 서비스 UART RX | 로그/점검용 |
| `PB0` | BL0942 ZX 입력 | 검증용 권장 |
| `PB1` | BL0942 CF 입력 | 검증용 권장 |
| `PC14` | LSE IN | RTC 필요 시 |
| `PC15` | LSE OUT | RTC 필요 시 |

## 4. 권장 이유

- `USART2 = BL0942`
- `USART3 = RS-485`
- `PA9/PA10`은 서비스 UART로 남겨 디버깅 여유 확보
- `PB0/PB1`은 BL0942 보조 신호 검증용으로 권장

## 5. 주의사항

- 최종 확정은 회로도와 PCB 배치 기준으로 조정해야 한다.
- CubeMX에서 최종 alternate function을 다시 확인한다.
