# HW팀 회로도 요청 체크리스트

작성일: 2026-04-02  
대상: HW팀 전달용

## 1. 회로도 요청 목적

단상 지선차단기 1차 제품용 회로도를 요청한다.

구성 기준:

- MCU: `STM32F107VCT6`
- 계량칩: `BL0942`
- 통신: `RS-485`
- 기능 범위: 단상 전력 계측 + 과전류 차단 + 상태/이벤트 전송

## 2. HW팀에 요청할 블록

- STM32F107 전원/클럭/리셋/디버그
- BL0942 전압/전류 입력부
- MCU-BL0942 통신부
- RS-485 트랜시버
- 릴레이/차단기 구동부
- 상태 LED
- 필요 시 비휘발성 저장부

## 3. MCU 쪽 필수 반영 항목

- `VDD/VSS`, `VDDA/VSSA` 전원 처리
- `NRST` 회로
- `BOOT0` 스트랩
- `SWDIO`, `SWCLK` 디버그 커넥터
- `HSE` 크리스탈 회로
- 필요 시 `LSE` 크리스탈 회로

## 4. BL0942 쪽 필수 반영 항목

- 전압 샘플링 입력 회로 `VP/VN`
- 전류 샘플링 입력 회로 `IP/IN`
- UART 통신 연결
- `SEL=UART` 설정
- `SCLK_BPS` baud 설정
- 필요 시 `CF/ZX` 출력 사용 여부 검토

## 5. 통신부 요청 항목

- RS-485 2선식
- MCU UART 연결
- DE/RE GPIO 제어
- 종단저항 / 바이어스 저항 검토
- 현장 커넥터 규격 반영

## 6. 보호/출력부 요청 항목

- 릴레이 구동 회로
- 역기전력 보호
- 릴레이 상태 피드백 여부
- 과전류 차단 시 fail-safe 동작 방식

## 7. HW팀에 같이 전달할 정책

- 1차 제품에서는 `누설전류/ZCT` 제외
- 1차 제품에서는 `고조파/아크` 제외
- 1차 제품은 `BL0942 기반 단상 전력 계측`이 핵심
- 펌웨어는 `STM32CubeMX + STM32 HAL` 기준으로 착수 예정

## 8. 회로도 수령 후 펌웨어팀이 바로 봐야 하는 것

- MCU 핀맵
- BL0942 연결 UART 번호
- RS-485 UART 번호
- 릴레이 제어 핀
- LED 핀
- 부트/리셋 회로
- 외부 크리스탈 주파수

## 9. 회로도 수령 후 필요한 산출물

- 최종 핀맵 표
- CubeMX IOC 초안
- BL0942 드라이버 핀 정의 헤더
- RS-485 드라이버 핀 정의 헤더
