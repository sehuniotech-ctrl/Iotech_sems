# MCU/계량칩 후보 검토

> 현재 문서는 초기 후보 검토 이력입니다. 최종 선정 MCU는 `STM32G071CBT6`이며, 본 문서의 `STM32F107` 검토 내용은 참고용 보관 문서입니다.

작성일: 2026-04-02  
대상: `STM32F107VCT6`, `BL0942`

## 1. 검토 기준

- 가격
- 게런티 / 단종 여부
- 사이즈

## 2. 단상 지선차단기 1차 요구 기능

필수 범위:

- 전압 RMS

- 유효전력

## 3. MCU 후보: STM32F107VCT6

### 3.1 요약 판단

- 종합 판단: `적합`

### 3.2 항목별 평가

| 항목 | 평가 | 내용 |
|---|---|---|
| 가격 | 보통 | LCSC 기준 1k 수량 약 `US$3.1769`, 1개 약 `US$4.7661` |
| 게런티/단종 | 양호 | ST 공식 페이지 기준 `Active`, `volume production`, ST 10년 longevity 프로그램 대상 계열로 판단 |
| 사이즈 | 보통 | `LQFP-100 14x14 mm`, 기능은 넉넉하지만 패키지는 큼 |
| 딜리버리 | 보통 | LCSC 재고 1000개, DigiKey 리드타임 13주 |
| 기능 구현 | 양호 | UART/SPI/GPIO/ADC/CAN/Ethernet/USB 등 자원 충분 |

### 3.3 장점

- STM32CubeMX 기반 개발 가능
- UART가 여러 개라 `BL0942 + RS-485` 분리 구성 가능
- GPIO 여유가 많아 릴레이, LED, 디버그, 향후 확장 대응 가능
- 산업 현장에서 검증된 STM32 계열이라 개발 자료가 많음

### 3.4 주의점

- 100핀 패키지라 보드가 커질 수 있음
- 현재 공식 페이지에는 Active로 표시되지만, 일부 유통처는 리드타임이 길다
- 이번 제품에 Ethernet/USB/CAN까지 모두 필요하지 않다면 오버스펙일 수 있다

### 3.5 결론

`STM32F107VCT6`는 기능적으로 충분하고 개발 착수도 수월하다.  
다만 크기와 가격은 최소 구성 MCU보다 불리할 수 있으므로, 이번 프로젝트에서는 `일정 안정성`과 `개발 편의성` 중심으로 선택하는 판단이 적절하다.

### 3.6 Longevity 메모

- ST는 공식 `Product Longevity` 프로그램을 운영한다.
- STM32는 ST 공식 안내 기준 `rolling 10-year commitment` 방식으로 관리된다.
- `STM32F107VCT6`의 exact part 행은 현재 수집된 페이지에서 직접 추출하지 못했지만, `STM32F107` 계열은 10년 프로그램 대상 계열로 판단된다.
- 최종 발주 전에는 제품 페이지의 `Quality and Reliability` 표에서 exact part 기준으로 다시 확인하는 것을 권장한다.

## 4. 계량칩 후보: BL0942

### 4.1 요약 판단

- 종합 판단: `1차 단상 제품에 적합`

### 4.2 항목별 평가

| 항목 | 평가 | 내용 |
|---|---|---|
| 가격 | 매우 양호 | LCSC 기준 1k 수량 약 `US$0.2248`, 1개 약 `US$0.3166` |
| 게런티/단종 | 보통 이상 | 공식 제품 페이지와 최신 데이터시트 존재, 다만 ST처럼 명시적 Active 표기는 확인 못함 |
| 사이즈 | 양호 | `TSSOP-14` 또는 `SSOP-10` |
| 딜리버리 | 양호 | LCSC 기준 재고 약 2902개, 즉시 출하 가능 표기 |
| 기능 구현 | 조건부 양호 | 단상 계량 필수 기능은 충족, 고급 보호 기능은 미충족 |

### 4.3 지원 기능

- 전류 RMS
- 전압 RMS
- 유효전력
- 유효전력량
- 빠른 전류 RMS
- 과전류 검出
- 주파수 측정
- 전압/전류 영점 통과 출력
- 샘플 파형 데이터 제공
- UART / SPI 통신

### 4.4 부족한 기능

- ZCT 기반 누설전류 직접 계측
- IGR / IGC 분리
- 고조파 분석
- 아크 전류 검출
- 내부 온도 측정

### 4.5 결론

`BL0942`는 단상 지선차단기 1차 범위에는 맞다.  
하지만 누설전류, 아크, 고조파까지 요구하면 칩 변경이나 별도 계측 경로가 필요하다.

## 5. 후보 조합 평가

### 5.1 STM32F107 + BL0942 조합

- 종합 판단: `현재 정의된 1차 제품 범위에는 적합`

가능한 구성:

- STM32F107: 메인 제어, RS-485, 릴레이 제어, 이벤트 처리, 저장, 상위 프로토콜
- BL0942: 전압/전류/전력/전력량/주파수 계량

### 5.2 이 조합으로 가능한 것

- 단상 전력 계측
- 과전류 차단
- 이벤트 로그
- RS-485 통신
- 서버 전송용 데이터 생성
- STM32CubeMX 기반 개발 착수

### 5.3 이 조합으로 바로 안 되는 것

- 누설전류 계측
- 고조파/아크 계측
- 절연열화 진단
