# HW팀 회로도 요청 체크리스트

작성일: 2026-04-03

## 1. 기준 조합

- MCU: `STM32G071CBT6`
- 계량칩: `BL0942`

## 2. 회로도 요청 목적

단상 지선차단기 1차 제품용 회로도를 작성하기 위한 입력 조건 정리

## 3. 필수 블록

- STM32G071CBT6 전원/리셋/디버그/클럭
- BL0942 전압/전류 입력부
- MCU-BL0942 UART 통신
- RS-485 트랜시버
- 릴레이 구동부
- 상태 LED

## 4. 회로도 반영 요청 사항

- `SWD`, `NRST`, `BOOT0`, `HSE` 살리기
- `USART2 = BL0942`
- `USART3 = RS-485`
- `PB12 = DE/RE`
- `PC6 = Relay Drive`
- `PC7 = Relay Feedback`
- `PC8/PC9 = 상태 LED`

## 5. 정책

- 1차 제품에서는 `ZCT`, `누설전류`, `고조파`, `아크` 제외
- 1차는 `단상 계량 + 과전류 차단 + RS-485` 중심

## 6. 회로도 수령 후 필요한 자료

- 최종 핀맵
- 크리스탈 주파수
- RS-485 트랜시버 부품
- 릴레이 드라이버 회로
- BL0942 스케일 값
