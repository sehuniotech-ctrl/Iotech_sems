# STM32U073C8T6 + BL0942 권장 핀맵 초안

작성일: 2026-04-06

## 1. 기준

- MCU: `STM32U073C8T6`
- 패키지: `LQFP48`
- 계량칩: `BL0942`
- 통신: `RS-485`
- 개발환경: `STM32CubeMX`

## 2. 기본 방향

- `USART1 = BL0942`
- `USART2 = RS-485`
- 릴레이, LED, DE/RE는 일반 GPIO로 배치
- 최종 Alternate Function은 `CubeMX`와 회로도 기준으로 다시 확인

## 3. 필수 핀

### 3.1 시스템

| 핀 | 기능 | 비고 |
|---|---|---|
| `VDD`, `VSS` | 디지털 전원 | 필수 |
| `VDDA`, `VSSA/VREF-` | 아날로그 전원 | 필수 |
| `PF2` | `NRST` | 외부 리셋 필수 |
| `PF3` | `BOOT0` | 풀다운 권장 |
| `PA13` | `SWDIO` | 디버그 필수 |
| `PA14` | `SWCLK` | 디버그 필수 |
| `PF0` | `OSC_IN` | 외부 HSE 사용 시 |
| `PF1` | `OSC_OUT` | 외부 HSE 사용 시 |

### 3.2 기능

| 핀 | 기능 | 비고 |
|---|---|---|
| `PA9` | BL0942 UART TX | `USART1_TX` 권장 |
| `PA10` | BL0942 UART RX | `USART1_RX` 권장 |
| `PA2` | RS-485 UART TX | `USART2_TX` 권장 |
| `PA3` | RS-485 UART RX | `USART2_RX` 권장 |
| `PA1` | RS-485 DE/RE | GPIO 출력 |
| `PB10` | Relay Drive | GPIO 출력 |
| `PB11` | Relay Feedback | GPIO 입력 |
| `PC6` | RUN LED | GPIO 출력 |
| `PC7` | TRIP/FAULT LED | GPIO 출력 |

## 4. 권장 핀

| 핀 | 기능 | 비고 |
|---|---|---|
| `PB0` | BL0942 ZX 입력 | 보조 검증용 권장 |
| `PB1` | BL0942 CF 입력 | 보조 검증용 권장 |
| `PB12` | 사용자 입력 또는 서비스 GPIO | 옵션 |
| `PB13` | 사용자 입력 또는 서비스 GPIO | 옵션 |
| `PB14` | 확장용 GPIO | 옵션 |
| `PB15` | 확장용 GPIO | 옵션 |
| `PC8` | 예비 LED 또는 상태 출력 | 옵션 |
| `PC9` | 예비 LED 또는 상태 출력 | 옵션 |

## 5. 권장 이유

- `PA9/PA10`은 BL0942 전용 UART로 두기 좋다.
- `PA2/PA3`은 RS-485 메인 UART로 구성하기 좋다.
- `PA1`은 `DE/RE` 전용 GPIO로 두면 통신 제어가 단순하다.
- `PB10/PB11`은 릴레이 제어와 피드백으로 분리하기 좋다.
- `PB0/PB1`은 BL0942의 보조 출력 검증 용도로 남겨두기 좋다.

## 6. CubeMX 시작 권장안

- `SYS -> Debug = Serial Wire`
- `RCC`
  - 초기 bring-up은 내부 클럭 우선
  - 필요 시 `PF0/PF1` 외부 클럭 사용
- `USART1`
  - `PA9 = TX`
  - `PA10 = RX`
  - 용도: `BL0942`
- `USART2`
  - `PA2 = TX`
  - `PA3 = RX`
  - 용도: `RS-485`
- `GPIO Output`
  - `PA1 = DE/RE`
  - `PB10 = Relay Drive`
  - `PC6 = RUN LED`
  - `PC7 = TRIP LED`
- `GPIO Input`
  - `PB11 = Relay Feedback`

## 7. 주의사항

- 본 초안은 `STM32U073C8T6`와 `BL0942`의 1차 제품 기준 권장안이다.
- 실제 핀 충돌 여부와 AF 선택은 `CubeMX`에서 최종 확인해야 한다.
- 회로도, PCB 배치, 절연거리, 통신 노이즈 조건에 따라 일부 GPIO는 조정될 수 있다.

## 8. 참고 근거

- `STM32U073C8T6`는 ST 공식 제품 페이지 기준 `LQFP48` 패키지이다.
- `STM32U073x8/B/C` 공식 데이터시트의 `LQFP48` 핀아웃에서 `PF2-NRST`, `PF3-BOOT0`, `PF0/PF1`, `PA2`, `PA3`, `PA9`, `PA10`, `PB0`, `PB1`, `PB10`, `PB11`, `PC6`, `PC7`, `PC8`, `PC9` 존재를 확인했다.
