# Smart Load 칩 선정 및 구매성

## 1. 최종 선정 칩

- MCU: `STM32L053R8T6`
- 계량칩: `ATM90E26-YU`

## 2. 선정 이유

- `STM32L053R8T6`
  - Smart Load 1차 기능 구현 가능
  - `STM32CubeMX` 사용 가능
  - 인셈 회신 기준 `Delivery ASAP`, `Price $1.10`, `MOQ 960 pcs`
- `ATM90E26-YU`
  - 무효전력 / 피상전력 / 역률까지 직접 계측 가능
  - 구매처 / 단가 / 리드타임 확인 완료

## 3. 후보 비교 핵심

| 항목 | STM32L053R8T6 | STM32U073RCT6 |
|---|---|---|
| 기능 구현 가능 여부 | 가능 | 가능 |
| 저전력 특성 | 유리 | 더 유리 |
| 가격 | `$1.10` | 소폭 더 유리 |
| Delivery | `ASAP` | `20 Weeks` |
| 조달 리스크 | 낮음 | 높음 |
| 최종 판단 | 선정 | 제외 |

## 4. 구매성 기준

- 최종 구매 판단은 가격만이 아니라 `Delivery`, `Buffer`, `공급성`을 우선한다.
- 기존 후보 문서는 `archive`로 이동 후 참고용으로만 보관한다.
