# ATM90E26 기능 전체 조사 및 확장 방향

작성일: 2026-04-07
대상 칩: `ATM90E26`

## 1. 목적

`ATM90E26`이 공식적으로 어떤 기능을 제공하는지 정리하고, 이 칩을 사용할 때 현재 지선차단기에서 어디까지 확장할 수 있는지 판단하기 위한 문서이다.

## 2. 칩 개요

- `ATM90E26`은 Microchip의 단상 고성능 전력 계량 AFE 칩이다.
- 공식 제품 페이지 기준 상태는 `In Production`이다.
- 단상 2선(`1P2W`), 단상 3선(`1P3W`), anti-tampering energy meter 용도로 설명되어 있다.
- 28-pin `SSOP` 패키지다.

## 3. 공식 자료 기준 핵심 기능

### 3.1 계량/측정 기능

- 전압 RMS 측정
- L line 전류 RMS 측정
- N line 전류 RMS 측정
- 평균 유효전력 측정
- 평균 무효전력 측정
- 평균 피상전력 측정
- 역률 측정
- 위상각 측정
- 주파수 측정
- 유효전력량 누적
- 무효전력량 누적
- 정/역방향 에너지 상태 확인
- 무부하 상태 확인

### 3.2 이벤트/상태 기능

- `Voltage Sag` 감지
- `WarnOut` 핀 출력
- `IRQ` 핀 출력
- `ZX` 영점 통과 출력
- 보정값 checksum 오류 감지
- active/reactive energy 방향 변경 상태 감지
- anti-tampering mode 지원

### 3.3 인터페이스/구성 기능

- `SPI` 또는 `UART` 인터페이스 선택 가능
- `CF1`: 유효전력 에너지 펄스 출력
- `CF2`: 무효전력 에너지 펄스 출력
- 외부 `8.192MHz` crystal 사용
- 전류 채널 gain 설정
- 전압 sag threshold 설정
- small-power mode 지원

## 4. 데이터시트 기준 직접 취득 가능한 항목

| 항목 | 직접 취득 여부 | 근거 |
|---|---|---|
| 전압 RMS | 가능 | `Urms` |
| 전류 RMS | 가능 | `Irms` |
| 유효전력 | 가능 | `Pmean` |
| 무효전력 | 가능 | `Qmean` |
| 피상전력 | 가능 | `Smean` |
| 역률 | 가능 | `PowerF` |
| 위상각 | 가능 | `Pangle` |
| 주파수 | 가능 | `Freq` |
| 유효전력량 | 가능 | active energy 계열 |
| 무효전력량 | 가능 | reactive energy 계열 |
| 무부하 상태 | 가능 | `Pnoload`, `Qnoload` |
| 정/역방향 상태 | 가능 | `RevP`, `RevQ` |
| 전압 sag 상태 | 가능 | `SagWarn` |
| 영점 통과 | 가능 | `ZX` 출력 |

## 5. 현재 지선차단기에서 바로 좋아지는 부분

`BL0942` 대비 `ATM90E26`으로 바꾸면 아래 항목이 좋아진다.

- 무효전력을 직접 레지스터 기반으로 다룰 수 있다.
- 피상전력을 MCU 추정이 아니라 직접 계측값 기준으로 다룰 수 있다.
- 역률을 직접 취득할 수 있다.
- 위상각을 직접 취득할 수 있다.
- N line current 채널이 있어서 anti-tampering 또는 추가 전류 감시 구조를 고민할 수 있다.
- `WarnOut`, `IRQ`, `ZX`, `CF1`, `CF2` 같은 하드웨어 보조 신호를 활용할 수 있다.

## 6. 확장 가능한 방향

### 6.1 전력 품질 기본 지표 확장

기존보다 아래 항목을 보다 자연스럽게 추가할 수 있다.

- 무효전력 표시
- 피상전력 표시
- 역률 표시
- 위상각 표시
- 정/역전력 방향 상태

즉, 단순 전압/전류/유효전력 계량기 수준에서 한 단계 확장된 `전력 상태 표시형 지선차단기` 구성이 가능하다.

### 6.2 이벤트/상태 감시 확장

아래 기능은 직접 차단 로직으로 바로 연결하지 않더라도 감시 기능으로 확장하기 좋다.

- 전압 sag 감시
- 보정값 이상 감시
- 영점 통과 기반 타이밍 제어
- 에너지 방향 변경 감시
- 무부하 상태 감시

활용 예:

- 전압 sag 발생 시 이벤트 로그 저장
- 영점 통과에 맞춘 릴레이 제어 최적화 검토
- 무부하 상태를 대기전력 차단 판단 보조 지표로 활용

### 6.3 펄스 기반 검증/보정 확장

`CF1`, `CF2` 출력이 있기 때문에 아래 활용이 가능하다.

- 생산 보정용 energy pulse 검증
- 시험 장비와의 펄스 비교
- 유효/무효 전력량 보정 확인

### 6.4 anti-tampering 응용 확장

공식 자료에는 anti-tampering mode가 있다.  
따라서 아래 같은 응용 가능성을 검토할 수 있다.

- L line / N line 전류 비교
- 전류 경로 이상 상태 감시
- 비정상 배선 또는 우회 사용 탐지 보조

다만 이것을 곧바로 `누전`이나 `영상전류` 기능으로 해석하면 안 된다.

## 7. 확장하기 어려운 항목

`ATM90E26`을 써도 아래는 여전히 별도 대응이 필요하다.

### 7.1 고조파 분석

- 데이터시트에는 THD, 차수별 harmonic, DFT 엔진 같은 직접 기능이 없다.
- 계량 정확도가 왜곡파에서도 유지되도록 설계된 것과, 고조파를 분석해서 내보내는 것은 다르다.
- 따라서 `고조파 발생현황 표시`까지 하려면 다른 계열 칩 또는 별도 신호처리가 필요하다.

### 7.2 누전/영상전류/저항성·용량성 누설전류

- `N line current` 채널이 있다고 해서 곧바로 `ZCT 기반 영상전류`를 대신하지는 못한다.
- 누전, Igr, Igc 같은 기능은 별도 `ZCT`와 위상 분석 경로가 필요하다.

### 7.3 아크전류

- 고주파 아크 검출 전용 기능이 없다.
- 아크 검출은 별도 아날로그 전처리와 알고리즘이 필요하다.

### 7.4 내부온도

- 내부 온도 센서 기반 계측 기능이 없다.
- 별도 센서 또는 MCU 내부 센서를 써야 한다.

## 8. 설계 관점에서 주의할 점

### 8.1 과전류/과전압 차단

`ATM90E26`은 계량칩이지 보호전용 차단칩은 아니다.  
따라서 아래 구조가 현실적이다.

- 전류/전압 측정: `ATM90E26`
- 임계값 판단: `MCU`
- 릴레이 차단: `MCU + 릴레이 구동부`

즉, 직접 차단보다 `측정 + MCU 보호판단` 구조로 이해하는 것이 맞다.

### 8.2 전압 sag 활용

`Voltage Sag`는 저전압 또는 순간 강하 감시에 유용하다.  
하지만 이것을 곧바로 과전압 보호로 확대 해석하면 안 된다.

### 8.3 UART/SPI 선택

- MCU 리소스와 보드 배선 복잡도에 따라 `UART` 또는 `SPI`를 선택할 수 있다.
- 현재 구조상 MCU와 다른 통신도 같이 써야 하므로 인터페이스 배치는 초기에 확정하는 것이 좋다.

## 9. 지선차단기 기준 확장 로드맵 제안

### 9.1 1차에서 바로 가능한 확장

- 전압
- 전류
- 유효전력
- 유효전력량
- 무효전력
- 피상전력
- 역률
- 주파수
- 과전류 판단
- 과전압 판단
- 전압 sag 이벤트
- 정/역방향 상태

### 9.2 2차에서 검토 가능한 확장

- 영점 통과 기반 릴레이 제어 최적화
- 무부하 상태와 대기전력 차단 로직 연계
- anti-tampering 응용 진단
- CF1/CF2 기반 생산/시험 자동화

### 9.3 별도 회로가 필요한 확장

- ZCT 기반 누전
- Igr / Igc
- 고조파 분석
- 아크 검출
- 내부온도 측정

## 10. 결론

`ATM90E26`은 `BL0942`보다 확장성이 더 좋다.  
특히 아래는 실질적인 장점이다.

- 무효전력 직접 측정
- 피상전력 직접 측정
- 역률 직접 측정
- 위상각 직접 측정
- 전압 sag / IRQ / WarnOut / ZX / CF1 / CF2 활용 가능
- N line current 기반 anti-tampering 응용 가능

반면 아래는 여전히 별도 대응이 필요하다.

- 고조파 분석
- 누전/영상전류
- Igr / Igc
- 아크전류
- 내부온도

따라서 `ATM90E26`은  
`전력 계량 + 전력 상태 표시 + 이벤트 감시` 쪽 확장에는 좋고,  
`누전/아크/고조파 진단형`까지 단독으로 해결하는 칩은 아니다.
