# BL0942 vs ATM90E26 비교표

작성일: 2026-04-07

## 1. 목적

기존 검토 칩 `BL0942`와 새 기준 칩 `ATM90E26`의 차이를 비교하여, 칩 변경 시 달라지는 기능과 확장 가능성을 정리한다.

## 2. 한 줄 요약

- `BL0942`: 단상 기본 계량에 강한 저가형
- `ATM90E26`: 단상 계량 + 전력 상태 분석 확장에 유리한 상위형

## 3. 기본 비교

| 항목 | BL0942 | ATM90E26 | 해석 |
|---|---|---|---|
| 칩 성격 | 단상 기본 계량칩 | 단상 고성능 계량 AFE | ATM90E26 쪽이 기능 폭이 넓음 |
| 패키지 | `TSSOP-14` | `SSOP-28` | ATM90E26 쪽이 핀 수와 크기 증가 |
| 인터페이스 | UART 기반 사용 검토 | SPI/UART 선택 가능 | ATM90E26 쪽이 유연함 |
| 전류 채널 | 1채널 중심 | L/N 2채널 | ATM90E26 쪽이 anti-tampering 응용 가능 |
| 에너지 펄스 | 보조 출력 사용 가능 | `CF1/CF2` 명확 지원 | ATM90E26 쪽이 생산/검증에 유리 |
| 보조 신호 | 제한적 | `WarnOut`, `IRQ`, `ZX` 지원 | ATM90E26 쪽이 상태 이벤트 처리에 유리 |

## 9. 칩 변경으로 달라지는 핵심

### 9.1 좋아지는 점

- 무효전력을 직접 지원한다.
- 피상전력을 직접 지원한다.
- 역률을 직접 지원한다.
- 위상각을 직접 지원한다.
- `WarnOut`, `IRQ`, `ZX`, `CF1`, `CF2`를 활용할 수 있다.
- 전압 sag 감시 기능을 활용할 수 있다.
- N line current 기반 추가 감시 응용이 가능하다.

### 9.2 주의할 점

- 패키지와 회로 구성이 커질 수 있다.
- MCU에서 하던 일부 계산은 줄지만, 칩 설정과 보정 절차는 더 신경 써야 한다.
- 누전, 아크, 고조파까지 되는 칩으로 오해하면 안 된다.

## 10. 결론

칩 변경의 핵심은 아래와 같다.

- `BL0942 -> ATM90E26`
- 기본 계량칩에서 상위 계량 AFE로 확장
- 전압/전류/유효전력 중심 구조에서 `무효전력/피상전력/역률/위상각`까지 직접 다루는 구조로 변경

즉, `ATM90E26`으로 바꾸면 `단상 전력 상태 표시형 지선차단기`로의 확장은 쉬워지지만, `누전/아크/고조파 진단형 지선차단기`가 바로 되는 것은 아니다.
