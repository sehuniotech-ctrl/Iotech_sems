# Smart Load용 ATM90E26-YU 기능 구분 및 핀 사용 정리

작성일: 2026-04-07
대상 칩: `ATMEL M90E26-YU`

## 1. 목적

`ATM90E26-YU`가 할 수 있는 기능을 아래 3가지로 구분한다.

- 전체 기능
- 현재 Smart Load에서 바로 사용할 기능
- 향후 확장 가능한 추가 기능

추가로, 기능과 연결되는 주요 핀 사용 목적도 같이 정리한다.

## 2. ATM90E26-YU 전체 기능

### 2.1 기본 계량 기능

- 전압 RMS 측정
- L line 전류 RMS 측정
- N line 전류 RMS 측정
- 유효전력 측정
- 무효전력 측정
- 피상전력 측정
- 역률 측정
- 위상각 측정
- 주파수 측정
- 유효전력량 누적
- 무효전력량 누적
- 정/역방향 에너지 상태 확인
- 무부하 상태 확인

### 2.2 상태/이벤트 기능

- 전압 sag 감지
- 보정값 checksum 오류 감지
- `IRQ` 인터럽트 출력
- `WarnOut` 경고 출력
- `ZX` 영점 통과 출력
- `CF1`, `CF2` 에너지 펄스 출력

### 2.3 인터페이스 및 설정 기능

- `SPI` / `UART` 선택 가능
- `MMD1`, `MMD0`로 metering mode 설정
- `USEL`로 UART/SPI 선택
- startup / no-load threshold 설정
- N line metering / anti-tampering mode 지원

## 3. 현재 Smart Load에서 바로 사용할 기능

현재 Smart Load 1차 기준으로는 아래 기능을 우선 사용하는 것이 적절하다.

### 3.1 직접 계측해서 사용할 기능

- 전압
- 전류
- 유효전력
- 유효전력량
- 무효전력
- 피상전력
- 역률
- 주파수

### 3.2 MCU가 판단해서 사용할 기능

- 과전류 판단
- 과전압 판단
- 릴레이 차단 판단
- 대기전력 차단 판단
- 이벤트 전송

### 3.3 보조적으로 활용 가능한 기능

- `IRQ`
  - 이벤트 발생 시 MCU 인터럽트 입력으로 활용 가능
- `WarnOut`
  - voltage sag 또는 보정 오류 감시용 가능
- `ZX`
  - 영점 통과 기준 제어 보조 가능

## 4. 향후 확장 가능한 추가 기능

### 4.1 전력 상태 확장

- 위상각 표시
- 정/역전력 방향 표시
- 전압 sag 이벤트 로그
- 무부하 상태 감시

### 4.2 생산/시험 확장

- `CF1`, `CF2` 펄스 기반 보정 확인
- 생산 시험 자동화
- 유효/무효 에너지 펄스 검증

### 4.3 운전/제어 확장

- `ZX` 기반 릴레이 제어 타이밍 최적화
- `IRQ` 기반 event-driven polling 최적화
- anti-tampering 응용 감시
- L line / N line 비교 기반 비정상 사용 상태 감시

## 5. ATM90E26-YU로도 바로 되지 않는 기능

아래는 별도 회로나 센서가 필요하다.

- 고조파 분석
- 누전차단
- 영상전류
- 저항성 누설전류
- 용량성 누설전류
- 아크전류
- 제품 내부온도

## 6. 주요 핀 사용 정리

### 6.1 전원 및 기준

| 핀명 | 핀 번호 | 용도 | 현재 사용 계획 |
|---|---:|---|---|
| `DVDD` | 3 | 디지털 전원 | 필수 |
| `DGND` | 2 | 디지털 그라운드 | 필수 |
| `AVDD` | 5 | 아날로그 전원 | 필수 |
| `AGND` | 6, 14 | 아날로그 그라운드 | 필수 |
| `Vref` | 13 | 기준 전압 출력 | 필수, 데이터시트 권장 디커플링 |
| `Reset` | 4 | 칩 리셋 | MCU 또는 reset 회로 연결 권장 |

### 6.2 계측 입력

| 핀명 | 핀 번호 | 용도 | 현재 사용 계획 |
|---|---:|---|---|
| `I1P` | 10 | L line 전류 + 입력 | 필수 |
| `I1N` | 11 | L line 전류 - 입력 | 필수 |
| `I2P` | 7 | N line 전류 + 입력 | 확장 검토 |
| `I2N` | 8 | N line 전류 - 입력 | 확장 검토 |
| `VP` | 16 | 전압 + 입력 | 필수 |
| `VN` | 15 | 전압 - 입력 | 필수 |

### 6.3 통신 및 설정

| 핀명 | 핀 번호 | 용도 | 현재 사용 계획 |
|---|---:|---|---|
| `USEL` | 12 | `UART/SPI` 선택 | 인터페이스 결정용, 고정 필요 |
| `CS` | 24 | SPI chip select | SPI 사용 시 필수, UART 사용 시 VDD 연결 |
| `SCLK` | 25 | SPI clock | SPI 사용 시 필수, UART 사용 시 GND 연결 |
| `SDO/UTX` | 26 | SPI data out / UART TX | MCU 통신 연결 |
| `SDI/URX` | 27 | SPI data in / UART RX | MCU 통신 연결 |
| `MMD1` | 1 | metering mode 선택 | 고정 설정 핀 |
| `MMD0` | 28 | metering mode 선택 | 고정 설정 핀 |

### 6.4 보조 출력

| 핀명 | 핀 번호 | 용도 | 현재 사용 계획 |
|---|---:|---|---|
| `WarnOut` | 17 | 치명 경고 출력 | 확장 권장 |
| `CF1` | 18 | 유효 에너지 펄스 | 시험/검증 확장 권장 |
| `CF2` | 19 | 무효 에너지 펄스 | 시험/검증 확장 권장 |
| `IRQ` | 20 | 인터럽트 출력 | 확장 권장 |
| `ZX` | 21 | 영점 통과 출력 | 확장 권장 |

### 6.5 클럭 및 기타

| 핀명 | 핀 번호 | 용도 | 현재 사용 계획 |
|---|---:|---|---|
| `OSCI` | 22 | 외부 크리스털 입력 | 필수 |
| `OSCO` | 23 | 외부 크리스털 출력 | 필수 |
| `Resv_Low` | 9 | reserved, GND 연결 | 필수 |

## 7. 현재 권장 핀 사용 방향

Smart Load 1차 제품 기준 권장 방향은 아래와 같다.

- `UART` 사용 시
  - `USEL = High`
  - `UTX/URX`를 MCU UART에 연결
  - `CS = VDD`
  - `SCLK = GND`
- 필수 연결
  - `DVDD`, `DGND`, `AVDD`, `AGND`, `Vref`
  - `I1P`, `I1N`, `VP`, `VN`
  - `OSCI`, `OSCO`
- 확장 고려 연결
  - `IRQ`
  - `WarnOut`
  - `ZX`
  - `CF1`, `CF2`
  - `I2P`, `I2N`

## 8. 실무 해석

- 현재 1차 Smart Load는 `전압/전류/유효전력/전력량` 수준을 넘어서 `무효전력/피상전력/역률`까지 직접 계측 구조로 확장 가능하다.
- 핀 수가 더 많아진 만큼, 처음부터 `보조 출력 핀`을 살려두면 향후 기능 확장에 유리하다.
- 특히 `IRQ`, `WarnOut`, `ZX`, `CF1`, `CF2`는 지금 당장 다 안 쓰더라도 보드에서 빼두는 것이 좋다.

## 9. 결론

`ATM90E26-YU`는 Smart Load에서 아래처럼 이해하면 된다.

- 전체 기능: 단상 고성능 전력 계량 + 상태/이벤트 보조 출력 + anti-tampering 응용
- 현재 사용할 기능: 전압, 전류, 유효전력, 무효전력, 피상전력, 역률, 전력량, 주파수
- 확장 가능한 기능: 위상각, 전압 sag, 정/역방향 상태, 영점 통과 기반 제어, 인터럽트 기반 이벤트 처리, energy pulse 기반 시험/보정
