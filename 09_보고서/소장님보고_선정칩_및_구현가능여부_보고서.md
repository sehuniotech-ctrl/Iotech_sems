# 선정 칩 및 구현 가능 여부 보고서

작성일: 2026-04-03

## 1. 보고 목적

초기 검토 단계에서 고려했던 `MAX` 계열 칩 및 기타 후보 검토를 정리하고, 현재 최종 선정한 칩 기준으로 구현 가능 여부를 보고드리고자 합니다.

## 2. 선정 결과

- 계량칩: `BL0942`
- MCU: `STM32G071CBT6`

## 3. 초기 검토 칩 정리

- `MAX` 칩은 초기 고려 대상이었으나 최종 선정에서 탈락했습니다.
- `STM32F107VCT6`는 후보 검토를 진행했으나 최종 선정 MCU는 아닙니다.
- 따라서 현재 이후 실무 기준은 `BL0942 + STM32G071CBT6`입니다.

## 4. 기능 구현 가능 여부

| 기능 | 구현 가능 여부 | 구현 주체 | 비고 |
|---|---|---|---|
| 전압 계측 | 가능 | BL0942 | 직접 계측 |
| 전류 계측 | 가능 | BL0942 | 직접 계측 |
| 유효전력 | 가능 | BL0942 | 직접 계측 |
| 유효전력량 | 가능 | BL0942 + STM32G071CBT6 | 누적/저장 관리 |
| 주파수 | 가능 | BL0942 | 직접 계측 |
| 과전류 차단 | 가능 | BL0942 + STM32G071CBT6 | MCU 보호 로직 |
| 차단기 상태 관리 | 가능 | STM32G071CBT6 | GPIO 처리 |
| RS-485 통신 | 가능 | STM32G071CBT6 | UART 사용 |
| 이벤트 로그 | 가능 | STM32G071CBT6 | 저장 정책 필요 |
| 피상전력/역률 | 가능 | STM32G071CBT6 | MCU 계산 |
| ZCT/누설전류 | 이번 1차 제외 | - | 별도 회로 필요 |
| 고조파/아크 | 이번 1차 제외 | - | 별도 분석 경로 필요 |

## 5. 결론

- 1차 제품 범위에서는 `BL0942 + STM32G071CBT6` 조합으로 구현 가능합니다.
- `MAX` 칩은 초기 고려했지만 탈락했습니다.
- 향후 HW 회로도 작성과 STM32CubeMX 기반 펌웨어 착수를 진행하면 됩니다.
