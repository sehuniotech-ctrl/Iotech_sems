# 단상 지선차단기 칩 선정 비교 보고서

작성일: 2026-04-02  
작성 목적: 단상 지선차단기 1차 개발용 MCU 및 계량칩 선정 방향 보고

## 1. 검토 배경

단상 지선차단기 1차 개발을 위해 계량칩 및 MCU를 선정할 필요가 있습니다.  
이번 검토는 아래 기능 범위를 기준으로 진행했습니다.

### 1.1 1차 제품 기능 범위

- 단상 전압 계측
- 단상 전류 계측
- 유효전력 계측
- 유효전력량 계측
- 주파수 계측
- 과전류 판단 및 차단
- 차단기 상태 관리
- RS-485 통신

### 1.2 1차 제외 범위

- ZCT 기반 누설전류
- IGR / IGC
- 고조파 분석
- 아크 검출

### 1.3 선정 기준

- 가격
- 게런티 / 단종 여부
- 사이즈
- 딜리버리
- 기능 구현 가능 여부

## 2. 기존의 칩을 썼을 때

기존 PCB 사진 분석 결과, 계량칩으로 확인 가능한 기존 칩은 아래와 같습니다.

- `STPM32`
- `ADE9153AACP`

사진상 확인 결과:

- `STPM32`: 계량칩 확인
- `ADE9153AACP`: 계량칩 확인
- 기타 칩: 계량칩으로 보기 어려움 또는 식별 불명확

### 2.1 기존 칩 사용 시 장점

- 기존 제품 자산 재활용 가능성
- 기존 회로 구조 일부 참조 가능
- 검토 이력이나 경험이 있을 가능성

### 2.2 기존 칩 사용 시 리스크

- 기존 회로/소스/자산 확보 상태가 불확실함
- 저장장치 복구 등 외부 변수에 의존하는 부분이 있음
- 부품 단가가 신규 후보보다 높음
- 현재 1차 제품 목표와 비교할 때 기능 과잉 가능성 있음

### 2.3 기존 계량칩 가격 비교

| 칩 | 제조사 | 계량칩 여부 | 1개 단가 참고 | 중량 수량 단가 참고 | 비고 |
|---|---|---|---:|---:|---|
| `BL0942` | Belling | 예 | `$0.3166` | `1,000+ $0.2248` | 신규 후보 |
| `STPM32TR` | STMicroelectronics | 예 | `$3.2843` | `1,000+ $1.9855` | 기존 보드 후보 |
| `ADE9153AACPZ-RL` | Analog Devices | 예 | `$10.35` | `100+ $7.01`, `5,000+ $6.02` | 기존 보드 후보 |

### 2.4 기존 칩 사용 검토 결론

- 기존 칩도 기능적으로는 사용 가능성이 있습니다.
- 다만 이번 1차 제품 범위에서는 `BL0942` 대비 가격 경쟁력이 크게 떨어집니다.
- 특히 `ADE9153AACP`는 기능은 우수하지만 단가 부담이 큽니다.
- 따라서 `기존 칩 유지`는 기술 검토보다 비용 및 일정 측면에서 불리합니다.

## 3. 기존의 칩과 다른 칩 비교

이번 검토에서는 기존 칩과 신규 후보를 함께 비교했습니다.

### 3.1 계량칩 비교

| 구분 | BL0942 | STPM32TR | ADE9153AACPZ-RL |
|---|---|---|---|
| 제조사 | Belling | STMicroelectronics | Analog Devices |
| 단상 1차 기능 적합성 | 적합 | 적합 | 적합 |
| 가격 | 가장 유리 | 중간 | 가장 높음 |
| 패키지 | TSSOP-14 | QFN-24 | LFCSP-32 |
| 공급성 | 양호 | 재고 적음 | 재고는 있으나 리드타임 부담 |
| 특징 | 저가, 기본 계량 적합 | 상위급 계량칩 | 고급 기능, 고가 |
| 종합 판단 | 1차 개발에 가장 적합 | 기능 대안 가능 | 단가 부담 큼 |

### 3.2 MCU 비교

현재 검토한 MCU는 아래와 같습니다.

- `STM32F107VCT6`
- `STM32G071CBT6`
- `STM32F103C8T6`
- `STM32C071CBT6`
- `STM32G0B1CCT6`
- `STM32F303CCT6`

| 구분 | STM32F107VCT6 | STM32G071CBT6 | STM32F103C8T6 | STM32C071CBT6 | STM32G0B1CCT6 | STM32F303CCT6 |
|---|---|---|---|---|---|---|
| 코어 | Cortex-M3 | Cortex-M0+ | Cortex-M3 | Cortex-M0+ | Cortex-M0+ | Cortex-M4F |
| Flash / SRAM | 256KB / 64KB | 128KB / 36KB | 64KB / 20KB | 128KB / 24KB | 256KB / 144KB | 256KB / 40KB |
| USART 수 | 최대 5 | 4 | 3 | 2 | 6 | 3 |
| 패키지 | LQFP-100 | LQFP-48 | LQFP-48 | LQFP-48 | LQFP-48 | LQFP-48 |
| 가격성 | 불리 | 양호 | 매우 양호 | 양호 | 보통 | 불리 |
| 공급성 | 보통 | 양호 | 매우 양호 | 재확인 필요 | 양호 | 양호 |
| CubeMX 사용 | 가능 | 가능 | 가능 | 가능 | 가능 | 가능 |
| Longevity | 계열 기준 확인 | 계열 기준 확인 | 직접 확인 | 계열 기준 확인 | 계열 기준 확인 | 직접 확인 |
| 종합 판단 | 기능은 충분하나 과함 | 가장 균형 좋음 | 저가형으로 적합 | 최소형 가능 | 확장형 가능 | 이번엔 과함 |

### 3.3 MCU 세부 검토 결과

#### STM32F107VCT6

- 기능적으로 충분
- UART/SPI/GPIO 자원 넉넉
- 단점은 `100핀`, 큰 보드 면적, 상대적으로 높은 원가
- 이번 1차 제품에는 다소 과한 구성 가능성

#### STM32G071CBT6

- 48핀
- USART 4개
- 메모리 적정
- CubeMX 지원
- 가격/기능/공급성 균형 우수
- 이번 프로젝트 기준 가장 적합한 대안

#### STM32F103C8T6

- 저가형 실무 대안
- 재고와 가격이 매우 좋음
- USART 3개로 `BL0942 + RS-485 + 서비스 UART` 구성 가능
- 메모리와 확장성은 G071보다 불리

#### STM32C071CBT6

- 최소 원가형으로 가능
- USART 2개로 여유가 적음
- 공급성은 발주 전 재확인 필요

#### STM32G0B1CCT6

- 기능 여유 매우 큼
- 이번 1차 제품에는 다소 과함

#### STM32F303CCT6

- 성능과 아날로그 기능은 우수
- BL0942를 별도 계량칩으로 쓰는 구조에서는 장점이 크게 살아나지 않음

## 4. 가장 추천되는 조합

### 4.1 계량칩 추천

`BL0942`

추천 이유:

- 단상 1차 기능 범위 충족
- 가격 경쟁력 가장 우수
- 패키지 작음
- 공급성 양호

### 4.2 MCU 추천 순위

1. `STM32G071CBT6`
2. `STM32F103C8T6`
3. `STM32F107VCT6`

추천 이유:

- `STM32G071CBT6`: 가격, 기능, 보드 크기, 통신 여유, longevity 측면에서 가장 균형이 좋음
- `STM32F103C8T6`: 저가형 대안으로 유리
- `STM32F107VCT6`: 기능은 충분하나 크기와 원가 면에서 비효율 가능

### 4.3 최종 권고 조합

#### 1안

- MCU: `STM32G071CBT6`
- 계량칩: `BL0942`

특징:

- 가장 균형 좋은 조합
- 1차 개발용으로 가장 적합

#### 2안

- MCU: `STM32F103C8T6`
- 계량칩: `BL0942`

특징:

- 저가형 대안
- 가격 중심 검토 시 유리

#### 3안

- MCU: `STM32F107VCT6`
- 계량칩: `BL0942`

특징:

- 기존 검토 방향 유지 가능
- 다만 보드 크기와 원가 측면에서 불리

## 5. 결론

### 5.1 결론 요약

- 기존 계량칩을 그대로 사용하는 방향은 기능적으로 가능하더라도 가격 및 일정 측면에서 불리합니다.
- 계량칩은 `BL0942`가 가장 적합합니다.
- MCU는 `STM32G071CBT6`가 가장 추천됩니다.
- 비용 중심 대안은 `STM32F103C8T6`입니다.
- `STM32F107VCT6`는 기존 후보이지만, 이번 1차 제품에는 다소 과할 수 있습니다.

### 5.2 최종 후보군 비교

|---|---|---|---|
| 계량칩 | BL0942 | STPM32TR | ADE9153AACPZ-RL |
| MCU | STM32G071CBT6 | STM32F103C8T6 | STM32F107VCT6 |
