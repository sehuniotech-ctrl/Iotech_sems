# 문서 운영 기준

작성일: 2026-04-08

## 1. 목적

Smart Load 문서는 매번 새 파일을 추가하는 방식이 아니라, 주제별 기준 문서를 계속 업데이트하는 방식으로 운영한다.

## 2. 운영 원칙

- 기준 문서는 주제별로 1개만 유지한다.
- 칩 변경, 일정 변경, 기능 변경이 생기면 기존 기준 문서를 수정한다.
- 중간 검토 자료, 후보 비교 자료, 이전 칩 기준 자료는 `archive`로 이동한다.
- 회의용 / 보고용 자료는 최신 기준 문서를 바탕으로 필요 시 생성한다.
- 기준 문서명에는 가능한 한 버전명이나 칩명을 넣지 않는다.

## 4. archive 운영

- 기존 칩 기준 문서
- 중간 검토용 문서
- 동일 주제의 구버전 문서
- 일회성 보고서 / 임시 산출물

위 문서는 삭제하지 않고 각 폴더의 `archive` 하위 폴더로 이동한다.

## 5. 현재 기준

현재 Smart Load 기준 칩은 아래와 같다.

- MCU: `STM32L053R8T6`
- 계량칩: `ATM90E26-YU`

따라서 이후 기준 문서는 모두 위 조합을 반영해 갱신한다.
