# STM32CubeMX 펌웨어 착수 준비

작성일: 2026-04-02

## 1. 전제

회로도 수령 전이라 실제 핀 번호는 미확정이다.  
따라서 지금 단계에서는 `CubeMX 프로젝트 구조`와 `필수 설정 항목`만 정리한다.

## 2. 펌웨어 착수 순서

1. HW 회로도 확인
2. 최종 핀맵 정리
3. STM32CubeMX IOC 생성
4. 클럭 설정
5. GPIO/UART 초기화
6. BL0942 통신 드라이버 작성
7. RS-485 통신 드라이버 작성
8. 계측값 변환 로직 작성
9. 과전류 판단/차단 로직 작성
10. 이벤트 로그 및 상태 전송 작성

## 3. CubeMX에서 먼저 설정할 항목

- Debug: SWD
- RCC: HSE 사용
- USART A: BL0942용
- USART B: RS-485용
- GPIO: 릴레이, LED, DE/RE
- NVIC
- 필요 시 RTC

## 4. 드라이버 구조 권장안

- `bsp_gpio`
- `bsp_uart`
- `drv_bl0942`
- `drv_rs485`
- `app_metering`
- `app_breaker`
- `app_event`

## 5. BL0942 드라이버 최소 기능

- 레지스터 읽기
- 체크섬 처리
- 전압 RMS 읽기
- 전류 RMS 읽기
- 유효전력 읽기
- 에너지 카운트 읽기
- 주파수 읽기
- 빠른 전류 RMS 읽기

## 6. 1차 펌웨어 목표

- 전원 인가 후 기동
- BL0942 값 정상 읽기
- RS-485로 측정값 송신
- 과전류 조건에서 릴레이 차단
- 상태 LED 표시

## 7. 필요한 입력 자료

- HW 회로도 PDF
- 핀맵 표
- BL0942 전압/전류 스케일 값
- 릴레이 동작 정책
- RS-485 프로토콜 정의
